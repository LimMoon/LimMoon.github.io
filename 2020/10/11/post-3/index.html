이번주엔 해커랭크 문제를 풀어보았다.

Common Child

해커랭크 코딩테스트
문자열 에서 한 개 또는 그 이상의 character를 제거해서 만들 수 있는 문자열을 child라고 할 때, 두 문자열의 가장 긴 common child의 길이는?

이 문제는 LCS(Longest Common Subsequence) 문제이다. LCS 문제는 Dynamic Programming을 이용해서 subproblem들로 나눠 풀 수 있다.

두 input string을 a,b 라고 하고, subproblem case를 정리해보자면,

  1. a[0] b[0]의 default 값은 0으로 둔다.
  2. a[i] == b[i] 일때, lcs(a[i-1],b[i-1]) + 1
  3. a[i] != b[i] 일때, max(lcs(a[i-1],b[i]), lcs(a[i], b[i-1]))

매트릭스를 이용해서 이 값을 a[0] b[0] 부터 찾아서 subproblem 결과를 합산한다. 합산할때는 비교해서 max를 킵하면 된다.

최초로 나타나는 같은 문자를 찾을 때 까지 문자열을 iterate 하면 된다. 예를 들어 COMPUTE 라는 문자열과 CMPT 가 있을때 아래 매트릭스처럼 채워나가서 iterate이 끝났을 떄의 맥스를 리턴하면 결과값이 나온다.
